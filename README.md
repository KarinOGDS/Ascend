# Ascend

**Ascend** is a gamified fitness tracking application inspired by RPG progression systems.

The goal of the project is to transform workout tracking into an engaging experience where users gain experience, level up, unlock achievements, and monitor their progress over time.

This project is also a long-term software engineering journey. It will evolve as new technologies are learned, growing from a simple JavaScript application into a complete full-stack product.

## Vision

Build a modern fitness application that combines:

* Workout tracking
* Personal records
* Experience points (XP)
* Levels
* Achievements
* Rankings
* Statistics
* Social features
